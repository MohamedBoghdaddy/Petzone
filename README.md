# PetZone

PetZone is a web-based pet care platform designed to help pet owners manage essential pet services in one place. The application provides tools for maintaining pet profiles, booking appointments, browsing pet products, and accessing veterinary-related support through a simple and user-friendly interface.

## About the Project

PetZone is designed to simplify the pet care experience by connecting pet owners with core services such as pet profile management, appointment booking, and product browsing.

The platform is built using HTML, CSS, and PHP, making it suitable for lightweight web deployment through local or hosted PHP servers. It is structured as a simple web application that can be extended with additional features such as authentication, online payments, veterinary dashboards, and product inventory management.

---

## Features

### Pet Profile Management

Users can add and manage pet profiles, including basic pet information and health-related records.

### Appointment Booking

Pet owners can schedule veterinary appointments or service bookings through the platform.

### Pet Product Store

Users can browse available pet products through the store section of the application.

### User-Friendly Interface

The platform provides a clean and simple interface that makes navigation easy for pet owners.

## Getting Started

To run PetZone locally, you need a PHP-supported local server environment.

### Prerequisites

Install one of the following local server environments:

- XAMPP
- WAMP
- MAMP
- Laragon

---

## Installation

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/Mostafaaelbarbary/PetZone.git
```

### 2. Move the Project Files

Move the cloned project folder into your local server root directory.

For XAMPP:

```bash
C:\xampp\htdocs\
```

For WAMP:

```bash
C:\wamp64\www\
```

### 3. Start the Server

Start Apache from your local server control panel.

### 4. Open the Application

Open your browser and visit:

```text
http://localhost/PetZone
```
